Introduction

Gamma and Beta functions are important mathematical functions used in calculus, physics, engineering, and probability theory.

History

The Gamma function was introduced by Leonhard Euler. It extends factorial values to real and complex numbers.

Important Formulas

Gamma Function

Γ(n) = (n - 1)!

Beta Function

B(x,y) = Γ(x)Γ(y) / Γ(x+y)
