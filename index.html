Gamma and Beta Function

Introduction

Gamma and Beta functions are important mathematical functions used in calculus, engineering, and physics.

History

The Gamma function was developed by Leonhard Euler. It extends the factorial function to real and complex numbers.

Formulas

Gamma Function

Γ(n) = (n-1)!

Beta Function

B(x,y)= Γ(x)Γ(y) / Γ(x+y)

Graphs

Solved Example

Find Γ(5)

Γ(5) = (5-1)! = 4! = 24
